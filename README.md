This is a simple 2D maze game. You play as a mouse trying to get a piece of cheese while a cat chases you.

The cat uses a greedy algorithm to track the mouse. The mouse can enter holes in the walls to stop the cat from chasing it. When the mouse leaves the walls, the cat will resume hunting it down.
