This is a simple 2D maze game. You play as a mouse trying to get a piece of cheese while a cat chases you. The maze is pre-determined.

The cat uses a greedy algorithm to track the mouse. The mouse can enter holes in the walls to stop the cat from chasing it. When the mouse leaves the walls, the cat will resume hunting it down.

### SET UP ###

You can download the exe application from the zip file in releases or here **insert itch io link*.

### CONTROLS ###

Move the mouse with arrow keys.

### WINNING ###

Win the game by reaching the cheese at the end of the maze. The game is over if the cat catches you.
